# 常见的选择器和标志以及权重值

```
id============标志：#  权重100
类============标志：.点  权重10

通配符=======标志：*，权重0
伪类========标志：一个冒号，权重10
```

# 盒子模型的组成

```
内容（content）   内间距(padding)   边框（border）   外间距（margin）
内间距和边框会撑大盒子
内间距不可以写负数
外间距可以写负数
padding:10px；======一个值是，上下左右都一样。
padding:10px 20px;======两个值，分别是上下  左右。
padding:10px 20px 30px;====三个值，分别是上  左右  下。
padding:10px 20px 30px 40px;====四个值，分别是：上右下左
外间距同理。
```

# 元素类型

```
块元素：block
	1.能加宽高，独占一行，竖着排列
	2.宽度默认和父亲同宽
	常见标签：div  p  ol  ul  li  dl  dt  dd  form等
行内元素：inline
	1.不能加宽高，横着排列
	2.宽度默认由内容撑开
	3.有时候上下间距可能加不上
	常见标签：span  i  em  b  strong  u  s  del  sup  sub等
行内块元素：inline-block
	1.能加宽高，横着排列
	2.默认有3-5px的间距
	常见标签：img  input  button
元素类型的转换：通过display转换
	display：block;=====转块
	display:inline;=====转行内
	display:inline-block;====转行内块
====================================================================
注意点：
	如果让你说3种类型：块、行内（内联）、行内块
  如果让你说2种：块、行内（内联）
```

# 定位模式和定位的特点

```
====================定位模式和特点====================
1.静态定位（默认定位，相当于没写），static
2.相对定位relative
		从文档流角度：不会脱离文档流（不脱标）
		有层叠性
		参照物：自己原来的位置
3.绝对定位absolute
		从文档流角度：会脱离文档流（全脱标）
		有层叠性
		参照物：有定位的祖先元素，如果都没有定位，那就是浏览器窗口的第一屏
4.固定fixed
		从文档流角度：会脱离文档流（全脱标）
		有层叠性
		参照物：浏览器窗口
5.粘性定位sticky
		不滚动是相对，滚动是绝对
====================================================================================================		
层叠性：意思就是元素的堆叠顺序，默认后面的会覆盖前面的，如果想手动调整可以通过z-index来调整。
====================================================================================================		
定位的主要作用是用来调整位置的，也可以实现覆盖，不同的效果使用的定位模式是不一样的。
可以通过偏移值来调整位置，也就是top bottom  left  right来调整位置。
一般覆盖效果用绝对
广告或者左右2侧导航用固定
吸顶用粘性
单穿的调位置可以用相对。

```

# BFC的特点和触发规则

```
1.BFC是一个独立的容器，子元素不会影响外面的元素。
2.BFC的区域不会和浮动的元素发生重叠（浮动的元素不会覆盖BFC的区域）====应用：左右2栏布局，双飞翼布局
3.计算BFC高度的时候浮动的元素也会参与计算：应用：高度塌陷问题（子元素浮动了，父元素没有高了）
4.属于同一个BFC的元素会发生外间距重叠
触发规则：
	1.html就是一个BFC
	2.浮动
	3.定位：绝对和固定
	4.overflow:hidden/auto/scroll
	5.display:inline-block /flex
```



# 宽高自适应

```
1.宽度自适应：
		a)宽度不写或者写width:auto;
		b)宽度写100%。
2.高度自适应：
		a)高度不写或者写height:auto;还可以写最小高：min-height。
		b)高度100%
目前写100%会出一个问题：加内间距或者边框会撑大盒子。
写最小高的好处：如果盒子没有内容或者内容比较少，也能保证这个区域是存在的，如果内容多，就随着内容撑开。
```



# 什么是高度塌陷

```
高度塌陷就是：子元素浮动了，或者定位了，就漂浮起来了，导致不占位置，撑不开父盒子，父盒子的高度受影响了。
解决方案：
	一：给塌陷的盒子写：overflow：hidden。缺点：如果有元素定位定到了盒子外面，那就会隐藏掉。
	二：在浮动的元素最后面写一个空标签，这个标签的主要目的是用来清除浮动的。缺点：增加了多余的标签，造成代码冗余。
	三:万能清除法：
		.clearfix::after{
         content:“”;
         display:block;
         width:0;
         height:0;
         overflow:hidden;
         visibility:hidden;
         clear:both;
    }
    写好之后给塌陷的盒子上加上这个类名即可。
```



# 什么是外间距重叠

```
兄弟关系：
	外间距不会累加求和，而是选择最大值去用。
父子关系：
	给子元素加上外间距，会把父元素带下来
	解决方式：
		1.让父亲触发BFC：overflow：hidden
		2.给父元素加边框线：border:1px solid transparent
```
